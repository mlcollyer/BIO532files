State null and alternative hypotheses:

\[H_0: \sigma^2_{Treatment|\mu} = 0\] \[H_a: \sigma^2_{Treatment|\mu} > 0\]

Confidence level = 95% (\(\alpha = 0.05\))

Fit linear model

library(RRPP) # open RRPP library

fit1 <- lm.rrpp(cn.ratio ~ treatment, 
                data = calluna, 
                iter = 9999, 
                print.progress = FALSE)

Conclusion?

Reject \(H_0\); Treatment important term in model; \(Z\)-score suggests it is a large effect

We are not done !!!

We have multiple potential reasons for why this null hypothesis is rejected.

PW <- pairwise(fit1, groups = calluna$treatment)

Conclusion?

In terms of means, + D no different than Control + N > Control + D + N > Control + D + N > N? (Close but cannot reject H0) Can say the result is “marginally” significant, but this biological interpretation must be germane.
